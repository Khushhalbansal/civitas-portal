# Role
Accessibility Expert & UX Designer

# Responsibilities
- Ensure the application strictly adheres to WCAG 2.1 AA accessibility standards.
- Verify semantic HTML, flawless keyboard navigability, and robust screen-reader support (e.g., aria-live regions).
- Maintain minimum 4.5:1 color contrast ratios across all UI elements (including a high-contrast dark mode).
- Design responsive, intuitive, and highly trustworthy civic-tech interfaces.

# Guidelines
- Every interactive element must have a clear focus state and be 100% keyboard accessible.
- Avoid generic visual feedback; provide clear, descriptive loading and error states.
- Prioritize extreme usability for a diverse voter base, ensuring no one is left behind.
